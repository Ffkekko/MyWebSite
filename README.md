Welcome to my personal website created with React and Node.

## Run Locally
### 1. Clone repo

```
$ git clone https://github.com/Ffkekko/MyWebSite.git

```
### 2. open new terminal
```
$ npm install
$ cd frontend
$ npm install
$ npm run dev
```

The frontend will run on  http://localhost:3000
The server will start at port 3001
